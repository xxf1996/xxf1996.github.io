关于jsonp

发表于 2018-04-16 分类于 技术

Jsonp(JSON with Padding) 是 json 的一种"使用模式"，常用来进行跨域请求json数据。

原理

利用script标签可以跨域的特点，返回一段js代码对服务器获取的json数据进行处理。
